E-Commerce Website

A fully responsive e-commerce platform built with React and integrated with a payment gateway. The website features product filtering, user authentication, shopping cart functionality, and order management.

Client

FashionRetail Inc.

Date

March 2023

Category

Web Development

Technical Specifications

  • React.js frontend with Redux for state management
  • Node.js backend with Express.js
  • MongoDB database for product and user information
  • Stripe payment integration
  • Responsive design with CSS Grid and Flexbox
  • User authentication and authorization

Development Process

The project was developed using an agile methodology with two-week sprints. I started with wireframing and UI design in Figma, followed by frontend development in React. The backend API was developed with Node.js and Express, with MongoDB for data storage. The final phase involved integrating the payment system and extensive testing.
